# Content Agent Insights from Larry (OpenClaw AI Influencer)

**Source**: Oliver Henry's article about Larry, an AI agent that got millions of TikTok views

## Key Capabilities Larry Has

1. **Persistent personality and memory** between conversations
2. **File system access** (read/write)
3. **Image generation** via OpenAI API
4. **Code execution** (writes code to add text overlays)
5. **Social media posting** via Postiz API
6. **Skill files** that teach specific workflows
7. **Memory files** where lessons learned are logged
8. **Research capabilities** to analyze winning formulas and trends
9. **Autonomous decision-making** (has opinions on what will work)

## Critical Content Agent Requirements

### 1. Research & Analysis Skills
- Ability to research winning content formulas
- Analyze competitor accounts
- Study platform-specific formats (e.g., TikTok slideshow formats)
- Spot trends on social platforms
- Learn from what's performing well

### 2. Content Creation Skills
- Generate hooks and captions
- Create images (via API)
- Add text overlays to images
- Write in the brand's voice
- Adapt content for different platforms

### 3. Workflow & Automation
- Schedule posts automatically
- Execute multi-step content workflows
- Manage entire content pipeline autonomously

### 4. Memory & Learning
- Persist lessons learned across sessions
- Build up knowledge over time via skill files
- Reference past performance to improve

### 5. Quality Control
- Evaluate content ideas (have opinions on what will work)
- Provide feedback on hooks and concepts
- Self-improve based on metrics

## Implications for TasteKit Content Domain

### Must-Have Questions in Onboarding
1. What platforms are you targeting? (TikTok, Instagram, Twitter, LinkedIn)
2. What type of content? (short-form video, images, text posts, threads)
3. What's your brand voice? (provide examples of posts you admire)
4. Who is your target audience?
5. What metrics matter to you? (views, engagement, conversions)
6. Do you want the agent to research trends autonomously?
7. What tools/APIs do you have access to? (image generation, video editing, scheduling)

### Must-Have Skills
1. **research-winning-formulas**: Analyze what's working in your niche
2. **generate-hooks**: Create attention-grabbing opening lines
3. **write-caption**: Generate platform-appropriate captions
4. **create-content-calendar**: Plan posts over time
5. **analyze-performance**: Review metrics and suggest improvements
6. **adapt-for-platform**: Reformat content for different social media
7. **generate-variations**: Create multiple options for user to choose from

### Must-Have MCP Tool Integrations
1. Image generation (DALL-E, Midjourney, etc.)
2. Social media scheduling/posting (Postiz, Buffer, etc.)
3. Analytics APIs (platform-specific metrics)
4. Web search for trend research
5. File system access for managing assets

## Key Insight

**The system around the agent is crucial.** It's not just about the AI model, but the skill files, memory logs, workflow automation, and tool integrations that make an effective content agent.
